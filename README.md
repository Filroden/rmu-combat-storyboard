## Welcome to RMU Combat Narrator

This module logs combat events and then provides the GM with a wizard to generate a prompt for an AI LLM to create a cinematic comic-style script of the combat and then generate the comic book illustrations. The module optimises the combat log for the LLM by removing redundant information and formatting it in a way that is easy for the LLM to understand. The module provides a user interface for the GM to select which combat events to include in the prompt and to customise the prompt before sending it to the LLM.

It is intented to be a fun tool to turn those epic combats into mini-comics that can be shared with players and keep those stories alive.

This module is designed for the Rolemaster Unified (RMU) system only.
